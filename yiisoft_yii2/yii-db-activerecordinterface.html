find() public method

Creates an ActiveQuery instance for query purpose.

This method is usually ment to be used like this:

Customer::find(1); // find one customer by primary key
Customer::find()->all(); // find all customers
yii\db\ActiveQueryInterface|static|null find( $q = null )
$q mixed

The query parameter. This can be one of the followings:

  • a scalar value (integer or string): query by a single primary key value and return the corresponding record.
  • an array of name-value pairs: query by a set of attribute values and return a single record matching all of them.
  • null (not specified): return a new yii\db\ActiveQuery object for further query purpose.
return yii\db\ActiveQueryInterface|static|null

When $q is null, a new yii\db\ActiveQuery instance is returned; when $q is a scalar or an array, an ActiveRecord object matching it will be returned (null will be returned if there is no matching).

link() public method

Establishes the relationship between two records.

The relationship is established by setting the foreign key value(s) in one record to be the corresponding primary key value(s) in the other record. The record with the foreign key will be saved into database without performing validation.

If the relationship involves a pivot table, a new row will be inserted into the pivot table which contains the primary key values from both records.

This method requires that the primary key value is not null.

void link( $name, $model, $extraColumns = [] )
$name string

The case sensitive name of the relationship.
$model static

The record to be linked with the current one.
$extraColumns array

Additional column values to be saved into the pivot table. This parameter is only meaningful for a relationship involving a pivot table (i.e., a relation set with [[ActiveRelationInterface::via()]].)

save() public method

Saves the current record.

This method will call insert() when \yii\db\isNewRecord is true, or update() when \yii\db\isNewRecord is false.

For example, to save a customer record:

$customer = new Customer; // or $customer = Customer::find($id);
$customer->name = $name;
$customer->email = $email;
$customer->save();
