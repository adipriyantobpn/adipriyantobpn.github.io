Class yii\rbac\Item

All Classes | Properties | Methods | Constants
Inheritanceyii\rbac\Item » yii\base\Object
Implementsyii\base\Arrayable
Available since version2.0
Source Code https://github.com/yiisoft/yii2/blob/master/framework/rbac/Item.php

Item represents an authorization item.

An authorization item can be an operation, a task or a role. They form an authorization hierarchy. Items on higher levels of the hierarchy inherit the permissions represented by items on lower levels. A user may be assigned one or several authorization items (called yii\rbac\Assignment assignments). He can perform an operation only when it is among his assigned items.
