Class yii\helpers\BaseConsole

All Classes | Methods | Constants
Inheritanceyii\helpers\BaseConsole
Subclassesyii\helpers\Console
Available since version2.0
Source Code https://github.com/yiisoft/yii2/blob/master/framework/helpers/BaseConsole.php

BaseConsole provides concrete implementation for yii\helpers\Console.

Do not use BaseConsole. Use yii\helpers\Console instead.
