ActiveRelation represents a relation between two Active Record classes.

ActiveRelation instances are usually created by calling yii\redis\ActiveRecord::hasOne() and yii\redis\ActiveRecord::hasMany(). An Active Record class declares a relation by defining a getter method which calls one of the above methods and returns the created ActiveRelation object.

A relation is specified by $link which represents the association between columns of different tables; and the multiplicity of the relation is indicated by $multiple.

If a relation involves a pivot table, it may be specified by via() or \yii\redis\viaTable() method.
